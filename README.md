# AI Policy Assistant (RAG)

A multilingual AI assistant that answers questions from government policy documents using Retrieval-Augmented Generation.

## Features
- Chat UI
- Hindi + English support
- Multiple PDFs
- Source citation

## Tech Stack
- Python
- Flask
- LangChain
- FAISS
- Hugging Face

## Setup

1. Install dependencies
pip install -r requirements.txt

2. Set API key
export HUGGINGFACEHUB_API_TOKEN=your_token

3. Run app
python app.py

4. Open
http://127.0.0.1:5000
